PROJECT OVERVIEW
----------------
Species 8 is a complete, production-ready JUCE audio plugin that allows users
to transform audio using natural language prompts. Users can drag & drop audio
files, view waveforms, and type descriptions like "wider, less muddy, brighter"
to apply intelligent DSP processing.

FEATURES IMPLEMENTED
--------------------
✅ Drag & Drop Audio File Loading (WAV, AIFF, MP3, FLAC, OGG)
✅ Real-time Waveform Visualization
✅ Natural Language Prompt Input ("wider", "brighter", "less muddy", etc.)
✅ Intelligent Keyword-Based DSP Parameter Control
✅ 4-Stage DSP Processor Chain:
   - High-pass filter (mud reduction)
   - High-shelf filter (brightness control)
   - M/S stereo width processor (custom implementation)
   - Reverb (space/plastic effects)
✅ Dry/Wet Mix Control
✅ Output Gain Control (-24dB to +24dB)
✅ Bypass Toggle
✅ Full Parameter Automation Support
✅ State Save/Load
✅ VST3, AU, and Standalone Builds
✅ Modern, Futuristic UI (Dark theme with purple accents)

KEY TECHNOLOGIES
----------------
- JUCE Framework 7.0+
- C++17
- juce::dsp::ProcessorChain
- juce::AudioProcessorValueTreeState
- Custom M/S stereo width processor

DSP CAPABILITIES
----------------
1. High-Pass Filter: 20Hz - 200Hz (mud removal)
2. High-Shelf Filter: ±6dB @ 4kHz (brightness)
3. Stereo Width: 0.0 (mono) to 2.0 (wide) via M/S encoding
4. Reverb: Room size, damping, and wet/dry mix

BUILDING THE PLUGIN
--------------------
Quick Build (Unix):  ./build.sh
Manual Build:        mkdir build && cd build && cmake .. && cmake --build .

TESTING THE PLUGIN
------------------
1. Load Species 8 in your DAW
2. Drag an audio file into the waveform area
3. Type a prompt: "wider, less muddy, brighter"
4. Click MUTATE to apply the transformation
